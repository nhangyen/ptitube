# Thư mục nhạc nền

Thêm file MP3 phi thương mại (Creative Commons / Royalty-Free) vào thư mục này.

Sau khi thêm, cập nhật `require()` trong `constants/MusicLibrary.ts`.

- https://incompetech.com/music/royalty-free/
- https://www.bensound.com/
- https://pixabay.com/music/
